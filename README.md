**lita-high-five** will make sure Lita doesn't leave you hanging.

## Installation

Add lita-high-five to your Lita instance's Gemfile:

``` ruby
gem "lita-high-five"
```

## Usage

```
You: Lita o/
Lita: \o
You: Lita \o
Lita: o/
```
